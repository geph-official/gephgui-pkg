This installer sets up Geph, the censorship-resistant VPN.

It will:

  • install Geph.app into your Applications folder, and
  • install a small background service that runs with administrator privileges so the VPN, kill switch, and proxy settings keep working across restarts — even when the app is closed.

You will be asked for your password once, to authorize the installation.

To remove Geph later, run sudo /Applications/Geph.app/Contents/Resources/uninstall.sh.
